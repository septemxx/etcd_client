# Changelog

## [0.1.0] - 2026-03-16

### 新增
- 初始化项目，基于 Tauri + React + TypeScript
- 实现连接管理功能，支持添加、编辑、删除连接配置
- 实现键值管理功能，支持增删改查操作
- 实现键的树形结构展示
- 支持连接测试功能
- 支持保存多个连接配置

### 技术栈
- 前端：React 19, TypeScript, Ant Design, Tailwind CSS, Vite
- 后端：Rust, Tauri, etcd-client, Tokio

### 功能详情
1. **连接管理**
   - 添加新连接配置
   - 编辑现有连接配置
   - 删除连接配置
   - 测试连接有效性
   - 保存连接配置到本地

2. **键值管理**
   - 查看所有键值对
   - 按前缀过滤键
   - 添加新键值对
   - 编辑现有键值对
   - 删除键值对
   - 复制键值

3. **用户界面**
   - 响应式设计
   - 树形结构展示键
   - 实时操作反馈
   - 简洁直观的操作流程
